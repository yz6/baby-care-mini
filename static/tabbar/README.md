# Tabbar 图标资源

已配置的 PNG 图标文件：

- `home.png` / `home-active.png` - 首页
- `service.png` / `service-active.png` - 服务
- `parent.png` / `parent-active.png` - 共育
- `mine.png` / `mine-active.png` - 我的

图标颜色配置：
- 未选中态：黑色主体
- 选中态：绿色主体 (`#2F8A5B`)

图标语义说明：
- 首页：房屋图标，表示工作台入口
- 服务：心形/关爱图标，表示服务管理
- 共育：人群/家庭图标，表示家长支持与共育
- 我的：用户头像图标，表示个人中心

## 文件格式说明

当前使用 PNG 格式图标文件，同时保留原始中文命名文件：
- `首页.png` / `首页-act.png`
- `服务.png` / `服务-act.png`  
- `共育.png` / `共育-act.png`
- `我的.png` / `我的-act.png`
